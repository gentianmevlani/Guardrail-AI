## Usage Quick Reference

```bash
# AI Code Explainer
npm run explain explain <file>              # Explain code
npm run explain diff <file1> <file2>        # Explain changes
npm run explain ask <file> <question>       # Ask about code

# Cross-Repository Intelligence
npm run cross-repo register <paths...>      # Register repos
npm run cross-repo learn                    # Learn patterns
npm run cross-repo best <pattern>           # Find best implementation

# Real-Time Code Quality Guardian
npm run guardian watch <file>               # Monitor in real-time
npm run guardian analyze <file>             # Analyze once
npm run guardian autofix <file>             # Auto-fix issues

# Dependency Impact Analyzer
npm run dep-analyzer analyze <dep>          # Analyze one dependency
npm run dep-analyzer all                    # Analyze all
npm run dep-analyzer plan                   # Generate update plan
```

## Integration Examples

### Example 1: Onboarding New Developer
```bash
# 1. Find authentication logic
npm run nl-search search "authentication"

# 2. Explain how it works
npm run explain explain src/auth.ts

# 3. See why it was built that way
npm run time-travel history src/auth.ts

# 4. Compare with other projects
npm run cross-repo best "authentication"
```

### Example 2: Major Refactoring
```bash
# 1. Start quality monitoring
npm run guardian watch src/api/users.ts

# 2. Check current implementation
npm run explain explain src/api/users.ts

# 3. Find similar code for patterns
npm run nl-search similar src/api/users.ts

# 4. Verify no production risks
npm run predict-prod
```

### Example 3: Dependency Update
```bash
# 1. Analyze update impact
npm run dep-analyzer analyze react 19.0.0

# 2. Generate update plan
npm run dep-analyzer plan

# 3. Check for issues
npm run predict-prod

# 4. Monitor during update
npm run guardian watch package.json
```
