# guardrail AI - Natural Language Examples

## 🎯 How to Use

guardrail AI now understands natural language! Just talk to it like you would a colleague.

## 💬 Example Commands

### Analysis & Review

```bash
# All of these work the same way:
guardrail analyze my project
guardrail check my code
guardrail review this
guardrail inspect
guardrail what's wrong
guardrail find issues
```

**Response:**
```
📊 I've analyzed your project and found some areas that could use attention.

Check the details above, or run "guardrail polish" to get specific fixes.
```

### Polish & Improvement

```bash
guardrail polish my project
guardrail improve my code
guardrail fix this
guardrail clean up
guardrail make it better
guardrail enhance
```

**Response:**
```
✨ I've polished your project and found 3 area(s) that could be improved.

Check the details above for specific recommendations.
```

### What's Missing?

```bash
guardrail what am i missing
guardrail what did i forget
guardrail check readiness
guardrail can i ship
guardrail am i ready
guardrail what's missing
```

**Response:**
```
🎯 I've checked your project for what AI app builders typically forget.

📊 SHIPPING READINESS SCORE
   🟡 65/100

⚠️ Not ready to ship - critical features missing
```

### Setup & Installation

```bash
guardrail setup
guardrail install
guardrail initialize
guardrail get started
guardrail configure
```

**Response:**
```
✨ Perfect! I've set up guardrail AI for your project.

Everything is configured and ready to go. You can now:
  • Run "guardrail analyze" to check your project
  • Run "guardrail polish" to find missing details
  • Run "guardrail vibecoder-check" to see what you might be missing
```

### Train ML Model

```bash
guardrail train model
guardrail learn
guardrail teach
guardrail build model
```

**Response:**
```
🧠 Great! I've trained the ML model on your codebase.

It has learned:
  • Your code patterns
  • Your conventions
  • Your architectural decisions
  • Your coding style

The model is now ready to provide intelligent, project-specific assistance!
```

### Platform Integration

```bash
guardrail install platform netlify
guardrail add platform supabase
guardrail setup platform vercel
guardrail deploy to railway
```

**Response:**
```
✅ Great! I've installed the netlify plugin for your project.

Configuration files have been created and you're ready to deploy!
```

### Help

```bash
guardrail help
guardrail what can you do
guardrail commands
guardrail how do i
```

**Response:**
```
🛡️ guardrail AI - Your AI coding companion that never drifts

I can help you with:

📊 Analysis & Review
  • "analyze my project" - Deep analysis of your codebase
  • "check my code" - Quick code review
  • "what's my status" - Project health check

✨ Polish & Improvement
  • "polish my project" - Find missing details
  • "improve my code" - Code quality improvements
  • "what am I missing" - Check for forgotten features

🧠 Learning & Setup
  • "train model" - Train ML on your codebase
  • "setup" - Complete project setup
  • "install platform netlify" - Add platform integration
```

### Status Check

```bash
guardrail status
guardrail health
guardrail how am i doing
guardrail what's my score
guardrail project status
```

**Response:**
```
📊 Your Project Status

✨ Polish Check: Completed
🎯 Vibecoder Check: Completed

💡 Run "guardrail analyze" for detailed insights
```

```bash
guardrail
```

## 💡 Tips

1. **Be Natural** - Just talk to it like a colleague
2. **Multiple Ways** - Many phrasings work for the same command
3. **Interactive Mode** - Run without args for a conversation
4. **Help Anytime** - Type "help" to see all commands
5. **Suggestions** - If unsure, guardrail will suggest commands

## 🚀 Quick Start

```bash
# Install globally
npm install -g @guardrail-ai/core

# Run in your project
cd my-project
guardrail analyze my project
guardrail what am i missing
guardrail polish
```
