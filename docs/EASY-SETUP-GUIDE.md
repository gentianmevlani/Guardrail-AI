## 🎯 For Users Who Don't Know About MCPs, CLIs, or Integrations

## 🚀 Super Simple Setup

### Option 1: Just Run It (Recommended!)

```bash
guardrail
```

That's it! guardrail AI will:
1. Detect your IDE automatically
2. Set up everything for you
3. Explain what it's doing in simple terms
4. Guide you through any steps that need your input

### Option 2: Conversational Setup

```bash
guardrail setup
```

Then just answer the questions or describe what you want:

```
What would you like to do?
1. Set up guardrail AI for my project
2. I want to build an AI app
3. I want to analyze my codebase
4. Just help me get started

Your choice: 1
```

### Option 3: Interactive Onboarding

```bash
guardrail onboard
```

This gives you a friendly, step-by-step guide that explains everything.

## 🎓 Understanding the Terms (Optional)

You don't need to know these, but here's what they mean:

### MCP (Model Context Protocol)
**Simple explanation:** It's like giving your AI assistant a map of your project. guardrail AI sets this up automatically - you don't need to do anything!

### CLI (Command Line Interface)
**Simple explanation:** It's just typing commands instead of clicking buttons. guardrail AI's CLI is conversational - you can talk to it naturally!

### Integration
**Simple explanation:** It's connecting guardrail AI to your IDE so everything works together. guardrail AI does this automatically!
