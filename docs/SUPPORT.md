## Before Asking for Help

To help us help you faster, please include:

1. **guardrail version:** `npx guardrail --version`
2. **Node.js version:** `node --version`
3. **Operating system:** Windows/macOS/Linux
4. **Command you ran:** The exact command
5. **Error message:** Full error output
6. **Configuration:** Your `guardrail.config.json` (if applicable)

## Self-Help Commands

```bash
# Check your environment
npx guardrail doctor

# Get version info
npx guardrail --version

# Get help for any command
npx guardrail <command> --help
```

## Security Issues

**DO NOT** report security vulnerabilities publicly.

Please email [security@guardrailai.dev](mailto:security@guardrailai.dev) instead.

See our [Security Policy](SECURITY.md) for more details.
